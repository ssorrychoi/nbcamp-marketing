## 1. 📈 핵심 광고 성과 지표 이해
광고 성과는 소비자의 행동 흐름에 따라 **노출 → 관심 → 성과** 단계로 나누어 측정합니다.

| 측정 단계 | 주요 지표 | 의미 | 실무 활용 포인트 |
| :--- | :--- | :--- | :--- |
| **노출 (Exposure)** | **Reach (도달률)** | 광고를 본 '순 유저 수' | 인지도 캠페인 관리, 광고 피로도(빈도) 조절 |
| | **Impression (노출수)** | 화면에 표시된 총 횟수(중복 포함) | 예산 대비 노출 효율 판단 |
| | **CPM** | 1,000회 노출당 비용 | 인지도 중심 캠페인의 매체 효율성 평가 |
| **관심 (Engagement)** | **CTR (클릭률)** | 노출수 대비 클릭 비율 | 소재 반응력 평가, 카피·이미지 A/B 테스트 |
| | **CPC** | 클릭 1회당 비용 | 클릭 유도형 광고 예산 효율 평가 |
| **성과 (Conversion)** | **CVR (전환율)** | 클릭수 대비 구매 등 행동 비율 | 랜딩페이지 타당성 및 CTA 최적화 판단 |
| | **CPA** | 행동(전환) 1회당 비용 | 전환 중심 캠페인의 성과 평가 핵심 지표 |
| | **ROAS (광고 수익률)** | 지출한 광고비 대비 발생한 매출 | 예산 효율성 및 채널별 성과 비교 |
| | **Retention (유지율)** | 유입된 유저의 재방문/재구매율 | 앱 및 CRM 마케팅의 장기 충성도 평가 |

> **🚨 실무 해석 포인트:**
> - CTR은 높으나 CVR이 낮다면? → 광고는 흥미로우나 랜딩페이지 설득력이 떨어짐.
> - CTR은 낮으나 CVR이 높다면? → 광고 타겟팅은 정확하나 크리에이티브(소재) 개선 필요.

## 2. 🎯 KPI 설정 원칙: SMART 프레임워크
KPI는 단순한 숫자가 아니라 조직의 전략적 방향성을 나타내는 **데이터 나침반**입니다.

- **S (Specific - 구체적):** 목표가 명확해야 함 (예: "앱 설치 3,000건")
- **M (Measurable - 측정 가능):** 수치로 확인 가능해야 함 (예: "CTR 2.5% 이상")
- **A (Achievable - 달성 가능):** 현실적인 목표여야 함 (예: "전월 대비 20% 증가")
- **R (Relevant - 관련성):** 비즈니스 최종 목표와 연결되어야 함 (예: "구매 전환 중심")
- **T (Time-bound - 기한):** 명확한 기간이 있어야 함 (예: "4주 내 달성")

> 💡 **참고: Outcome vs Output**
> - **Output (산출물):** "무엇을 했는가?" (예: 광고 캠페인 10개 세팅)
> - **Outcome (성과):** "그래서 무엇이 달라졌는가?" (예: 신규 고객 유입률 20% 증가) → **마케터는 Outcome에 집중해야 합니다.**

---

## 3. 📝 KPI 달성 전략 시트 작성
전략과 실행을 데이터로 연결해 주는 브릿지 문서입니다. "전환을 늘리자"는 추상적인 목표를 "CVR을 2.5%에서 3.3%로 올리자"라는 **측정 가능한 언어로 번역**해 줍니다.

- **작성 항목:** 캠페인 목표, KPI, 기준값(Base), 목표값(Target), 기간, 실행 전략(Action), 측정 도구(Tool), 성과(Result), 인사이트(Insight).
- **장점:** 캠페인의 성과를 정량적으로 비교할 수 있으며, 성과가 오르거나 떨어진 **원인(Insight)**을 구조적으로 분석하는 사고 훈련이 가능합니다.

## 4. 🧩 미디어 믹스 (Media Mix) 설계
단일 매체의 성과가 아닌, **목표 달성을 위해 여러 매체를 조합하여 시너지를 극대화하는 예산·채널 배분 전략**입니다.

- **미디어 믹스 설계 프로세스:**
  1. **KPI 선정:** SMART 원칙에 따라 구체적 KPI 설정 (예: CPA 9,000원 이하).
  2. **채널 목적 정의:** 퍼널에 맞춰 채널별 역할 부여 (예: 인지=TikTok, 전환=Naver 검색).
  3. **예산 배분:** 캠페인 목적에 맞춰 예산 분할 (예: 신규유입 40% / 전환 40% / 리텐션 20%).
  4. **미디어 믹스 테이블 작성:** 매체명, 예산, 예상 노출/클릭/전환수, CPA, ROAS 등을 한눈에 구조화.
- **AI 활용 팁:** 과거 데이터가 없는 신규 런칭 캠페인의 경우, ChatGPT 등을 활용하여 업계 벤치마크 기반의 예상 CTR, CVR, CPM 등을 시뮬레이션하고 미디어 믹스 초안을 빠르게 작성할 수 있습니다.
